Текстовое описание решения.
1) Для решения этой задачи в первую очередь позволяю пользователю ввести число элементов в массиве.
- Если пользователь ввел не число - сообщаю ему об этом и выхожу из программы.
- Если пользователь ввёл число - вызываю метод в котором пользователь вводит каждый элемент, записываю их в массив.
2) Затем пробегая циклом по массиву, проверяю каждый элемент массива и нахожу элементы, длина которых больше или равна указанной максимальной длине.
- Если длина элемента удовлетворяет условию - добавляю элемент в список незаданной длины. 
- Если не удовлетворяет - перехожу к следующему элементу массива.
3) Перевожу полученный список в массив методом ToArray().
4) Вывожу получившийся массив строк из трех и менее элементов.
